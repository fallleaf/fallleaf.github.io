博文的分类方法

2026年1月8日
阅读时长: 3 分钟

写博客写久了，会发现一个问题：文章越多，越需要一个 清晰、长期有效、边界明确 的分类体系。 分类不是为了把文章塞进抽屉，而是为了让写作更自由，让阅读更轻松，让未来的自己更容易找到过去的思考。

经过不断整理，我最终确定了五个维度，分别对应我生活与思考的五个方向：

  • 成长与学习
  • 技术与实践
  • 生活与日常
  • 行走与体验
  • 随笔与观察

它们构成了我博客的基础结构，也构成了我记录世界的方式。

为什么需要分类体系

分类体系的意义不在于“管理文章”，而在于：

  • 帮助我在写作前快速定位主题
  • 帮助读者理解文章的语境
  • 帮助未来的自己检索、回顾、复盘
  • 帮助博客在长期演化中保持结构稳定

一个好的分类体系应该 边界清晰、互不重叠、十年不过时。

我的五大分类

成长与学习（Growth）

核心定义： 关于个人能力、认知、方法论、职业成长的内容。 本质是“让自己变得更好”。

涵盖内容：

  • 学习方法、学习系统
  • 认知提升、思维方式
  • 工作方法、经验总结
  • 阅读笔记（偏方法论）
  • 复盘与成长记录

判断标准： 如果文章的核心是 提升自我能力 / 思维 / 方法 → 属于这里。

技术与实践（Tech & Making）

核心定义： 关于技术、工程、工具、系统搭建、实操经验。 本质是“解决问题、构建系统”。

涵盖内容：

  • 通信工程、网络、设备
  • 技术原理、技术解释
  • 工具使用、配置、脚本
  • 技术方案、架构设计
  • 实操经验、踩坑记录

判断标准： 如果文章围绕 技术本身 或 技术实践 展开 → 属于这里。

生活与日常（Daily Life）

核心定义： 关于日常生活、家庭、家务、植物、水培、生活方式改善。 本质是“生活本身”。

涵盖内容：

  • 家庭、亲子、生活记录
  • 水培、植物、家居
  • 生活方式、习惯、整理
  • 饮食、日常观察

判断标准： 如果文章主要描述 生活本身 → 属于这里。

行走与体验（Outdoor & Travel）

核心定义： 关于旅行、城市印象、户外体验、走过的地方。 本质是“在路上”。

涵盖内容：

  • 城市印象
  • 旅行记录
  • 户外体验
  • 旅途见闻
  • 地方文化

判断标准： 如果文章描述 地点 / 行走 / 体验 → 属于这里。

随笔与观察（Essays & Thoughts）

核心定义： 关于思考、观察、抽象感受、非主题性的文字。 本质是“表达与观察”。

涵盖内容：

  • 随笔、短文
  • 抽象思考
  • 观察、感受
  • 不属于其他分类的内容

判断标准： 如果文章难以归入前四类，或更像“写给自己看的文字” → 属于这里。

分类的判断方法

写文章前问自己一句：

这篇文章的“核心动作”是什么？

  • 提升自己 → 成长与学习
  • 解决技术问题 → 技术与实践
  • 记录生活 → 生活与日常
  • 描述旅途 → 行走与体验
  • 表达感受 → 随笔与观察

只要抓住“核心动作”，分类永远不会混乱。

分类体系的设计原则

  • 边界清晰：每类都有明确的核心动作
  • 互不重叠：同一篇文章不会在两个分类之间摇摆
  • 长期有效：十年后仍然适用
  • 可扩展：未来可以在每类下继续细分标签
  • 与写作习惯一致：符合我记录生活与思考的方式

未来可能的扩展

未来我可能会为每个分类增加：

  • 专属封面插画
  • 分类色彩体系
  • 分类首页介绍
  • 相关标签的推荐结构
  • 更细粒度的子分类（如必要）

但五大分类本身会保持稳定。

结语

分类不是限制，而是帮助我更清晰地理解自己。 它们像五个方向，让我在写作时知道自己正在走向哪里，也知道每篇文章属于哪一块土地。

未来的文章会继续在这五个维度中生长，而我也会在写作中不断更新它们。
